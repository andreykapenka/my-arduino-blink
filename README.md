\# Arduino Blink \& SOS Project



Учебный проект для освоения Git в разработке под Arduino.



\## Оборудование

\- Arduino Uno

\- 2 светодиода (пины 13 и 12)

\- Резисторы 220 Ом

\- Макетная плата



\## Функциональность

1\. Основной режим: попеременное мигание светодиодов

2\. Режим SOS: сигнал SOS светодиодом на пине 13
